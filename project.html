üåü Smart Street Lighting System Using Arduino and LDR

üîç Overview

This project is a simple and efficient Smart Street Lighting System using Arduino and an LDR (Light Dependent Resistor). The primary goal is to automatically control street lights based on the intensity of natural light, reducing energy consumption and manual effort.

üéØ Objective

To build an automatic street light system that turns ON when it gets dark and turns OFF during the day using an LDR and Arduino.

üß† How It Works

The LDR is a sensor that detects the presence of light. When the environment becomes dark (low light), the resistance of the LDR increases, signaling the Arduino to turn ON the street light. When daylight is present, the resistance decreases, and the Arduino turns the light OFF.

üíª Arduino Code


int ldrPin = A0;       // LDR connected to analog pin A0
int lightPin = 13;     // LED or Relay connected to digital pin 13
int threshold = 500;   // Adjust based on environment

void setup() {
  pinMode(lightPin, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  int ldrValue = analogRead(ldrPin);
  Serial.println(ldrValue);

  if (ldrValue < threshold) {
    digitalWrite(lightPin, HIGH);  // Turn ON light
  } else {
    digitalWrite(lightPin, LOW);   // Turn OFF light
  }

  delay(500);  // Adjust delay as needed
}
